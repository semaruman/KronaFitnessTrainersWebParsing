# Парсинг тренеров "Крона Фитнесс" Чебоксары

> Цель: ознакомление в области веб-парсинга

Консольное приложение на C# для парсинга информации о тренерах тренажёрного зала **"Крона Фитнес" (Чебоксары)** с официального сайта.

## 🚀 Возможности

- Парсинг списка тренеров (имя, ссылка на профиль)
- Вывод данных в консоль в удобном табличном формате

## 🛠 Технологии
| Технология/библиотека | Назначение |
|------------|------------|
| **.NET 10** | Среда выполнения |
| **HtmlAgilityPack** | парсинг HTML |
| **Selenium.WebDriver, Selenium.WebDriver.ChromeDriver** | эмуляция действий пользователя в браузере |
| **System.Xml.Linq** | парсинг XML |
